# Perbandingan Harga Rumah dengan UMP di Indonesia

Aplikasi berbasis Streamlit untuk menganalisis perbandingan harga rumah
dengan Upah Minimum Provinsi (UMP) di Indonesia.

## Struktur Folder

- Scraping : pengambilan data harga rumah
- Cleaning : data cleaning & preprocessing
- Data     : data mentah dan data bersih
- Visualisasi : grafik analisis
- GIS : peta interaktif
- Pages : halaman Streamlit

## Cara Menjalankan

pip install -r requirements.txt  
streamlit run app.py
